# 💼 Naveen Kumar A — Personal Portfolio

A professional, dark-themed personal portfolio website showcasing my identity as an **AI/ML Developer**, **Designer**, and **Developer**.

---

## 🎯 Objective

To design and develop a modern, responsive, single-page portfolio website that presents my professional profile, technical skills, featured projects, education, certifications, and contact information — all within a visually striking, premium dark-themed interface.

---

## 🛠️ Tools & Technologies Used

| Category         | Technology                                                 |
| ---------------- | ---------------------------------------------------------- |
| **Structure**    | HTML5 (semantic sections, SVG icons, Canvas element)       |
| **Styling**      | Vanilla CSS (custom properties, glassmorphism, animations) |
| **Interactivity**| Vanilla JavaScript (ES6+, Canvas API, Intersection Observer) |
| **Typography**   | Google Fonts — *Inter*, *JetBrains Mono*                   |
| **Icons**        | Lucide Icons (via CDN), inline SVGs for GitHub & LinkedIn  |
| **Design**       | Dark theme, glassmorphism, gradient accents, particle effects |

---

## 📋 Steps Performed

1. **Structured the page layout** — Built a single-page architecture with six clearly defined sections: Hero, About, Skills, Projects, Education, Certifications, and Contact, each with numbered section labels (01–06).

2. **Created the Hero section** — Designed a full-viewport landing area with an availability badge, gradient-styled name heading, role tags (AI/ML Developer • Designer • Developer), a tagline, call-to-action buttons, and social media links.

3. **Built the About section** — Added a personal introduction with stat cards displaying key metrics (2+ Projects, 5+ Certifications, 3+ Tech Domains, Expected Graduation 2028) with animated counters.

4. **Designed the Skills section** — Organized skills into four categories (AI/ML, Development, Design, IoT & Hardware) using a card-based grid layout with icon headers and tag-style skill badges.

5. **Showcased Featured Projects** — Created project cards for **SIGNORA** (AI-powered real-time sign language translator) and **MINO MINE** (IoT smart safety helmet for mining workers), each with project images, tech stacks, descriptions, and GitHub links.

6. **Added Education timeline** — Built a vertical timeline card showing B.E CSE (AIML) at Sri Sai Ram Institute of Technology (2024–2028).

7. **Listed Certifications** — Displayed five certifications (App Development Internship, Java Programming, Joy of Computing, Learning Analytics Tools, Building AI Agents with MongoDB) in a responsive grid with Lucide icons.

8. **Built the Contact section** — Added clickable contact cards for Email, LinkedIn, and GitHub, along with a downloadable resume button.

9. **Implemented the particle background** — Used the Canvas API to render 80 animated particles with connecting lines between nearby particles, creating a dynamic ambient network effect.

10. **Added cursor glow effect** — Implemented a radial glow that smoothly follows the mouse cursor on desktop (disabled on mobile for performance).

11. **Built scroll-triggered animations** — Used Intersection Observer to trigger staggered fade-in animations on elements as they enter the viewport.

12. **Designed the navigation** — Created a fixed, glassmorphic navbar with active-section highlighting based on scroll position, and a hamburger toggle for mobile navigation.

13. **Made it fully responsive** — Added CSS media queries for multiple breakpoints to ensure smooth rendering across desktop, tablet, and mobile screens.

14. **Applied glassmorphism styling** — Used `backdrop-filter: blur()`, semi-transparent backgrounds, subtle borders, and layered shadows across all cards for a frosted-glass aesthetic.
